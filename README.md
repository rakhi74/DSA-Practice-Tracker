# DSA Practice Tracker

A modern React + Vite web application for tracking Data Structures & Algorithms practice progress.

## Features

- 📊 Dashboard with statistics
- ➕ Add problems with validation
- 📋 Problems table with search & sort
- 📈 Topic & Difficulty analytics
- 🎯 Goal tracker
- 🔥 Streak system
- 🕐 Recent activity
- 💼 Interview readiness score
- 💾 Export/Import (JSON & CSV)
- 🌙 Dark mode
- 📱 Responsive design
- 🎖️ Achievement badges
- 📅 GitHub-style heatmap calendar

## Tech Stack

- React 18
- Vite
- Recharts (for charts)
- LocalStorage (for persistence)

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy the `dist` folder to Vercel, Netlify, or any static hosting.
